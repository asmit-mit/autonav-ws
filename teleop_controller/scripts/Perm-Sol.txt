For a more permanent solution, you can create a udev rule that sets the permissions automatically when the device is connected. 
Create a file in /etc/udev/rules.d
File Name: 777-ttyAcm0.rules
Content:
KERNEL=="ttyACM0", MODE="0777"


After Adding the Rules, Relaod udev:
sudo udevadm control --reload-rules
sudo udevadm trigger
